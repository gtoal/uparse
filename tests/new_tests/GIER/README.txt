These files are from https://datamuseum.dk/wiki/GIER/Turbo_GIER_ALGOL_4 and have been converted
from PDF format to Unicode by the utilities in the 'build' subdirectory here.

Only the _10 character has not been rendered accurately, because the Unicode character for that
symbol has not yet been universally implemented and therefore does not always render well.  For
the moment the symbol '@' has been substituted, which is not in any way ambiguous in these files.
